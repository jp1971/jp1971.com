== Description ==

Easily share documentation of site features with a "docs" custom post type. A capabilities check limits who can view docs on the front end. By default, admins can create docs and admins and editors can view them.

== Installation ==

1. Install through the WordPress admin or upload the plugin folder to your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
